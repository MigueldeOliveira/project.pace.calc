
# 🏃‍♂️ Pace Calc - Dashboard de Treinos

Aplicação web interativa para cálculo e acompanhamento de treinos, com autenticação via Google e dashboard personalizado com dados salvos na nuvem.

---

## 🚀 Sobre o Projeto

O Pace Calc é uma aplicação desenvolvida com foco em atletas e praticantes de atividade física que desejam acompanhar sua evolução de forma simples e eficiente.

A aplicação permite registrar, visualizar e analisar dados de treinos em um ambiente intuitivo, criando um histórico completo de desempenho individual.

---

## 🔐 Funcionalidades

### 👤 Autenticação
- Login com Google
- Exibição de nome e avatar do usuário
- Logout via menu dropdown

### 📊 Dashboard Personalizado
- Quantidade total de treinos
- Quilometragem acumulada
- Calorias totais
- Histórico completo de atividades

### 🧠 Controle de Treinos
- Salvamento de treinos diretamente nas calculadoras
- Visualização de histórico individual

### 🎯 Metas
- Criação de metas com prazo
- Marcação de metas como concluídas

### ⚖️ Acompanhamento físico
- Registro de peso ao longo do tempo

## ⚙️ Configuração do Firebase

Para que o login funcione corretamente em produção:

1. Acesse o Firebase Console  
2. Vá em Authentication → Settings  
3. Em Authorized domains, adicione:

seu-projeto.vercel.app
